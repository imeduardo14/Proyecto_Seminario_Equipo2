# Sistema WEB de Bitacoras/Visitas
El objetivo del proyecto es transformar una bitácora tradicional en papel en un sistema web digital, el cual permitirá registrar, consultar, editar y almacenar actividades realizadas dentro de la
institución.

# Funcionalidades del sistema
• Inicio de sesion: Controlará el acceso mediante sesiones en PHP.

• Registro Digital de Bitácoras: Captura de datos (Nombre, Campus, Fecha, Control de horas, Objetivo, Actividades, Resultados, Notas y Firmas).

• Firma Digital: Area de firma donde se almacenara digitalmente (Mediante mouse, touchpad o pantalla tactil).

• CRUD: Administracion de informacion (Registro, Visualizacion, Edicion y Baja de registros).

• Generación de PDF: Permite guardar e imprimir la bitácora en formato digital PDF.

•Base de Datos: Almacena y organiza toda la información y firmas de las bitácoras.

# Herramientas de desarrollo
• Frontend: HTML, CSS, JavaScript.

• Backend: PHP.

• Base de datos: MySQL.

• Otros: Signature Pad.js (esencial para firmas digitales).
